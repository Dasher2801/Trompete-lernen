# Trompete-lernen
Master Trumpet Fingerings: The Interactive Trainer
This minimalist web app is the perfect digital tool 
for trumpet beginners, music students, and returning playersto intuitively master classic
fingerings in the treble clef.Without any downloads or registrations, the tool transforms 
any web browser into a motivating educational game.

The app randomly generates notes between low C and high C,
featuring a dynamic sheet music staff that automatically handles ledger lines.
You control the virtual valves using your keyboard keys 1, 2, and 3, while pressing
0 immediately resets all valves to the open position.

Pressing the Enter key evaluates your attempt. If you are correct,
the entire screen instantly turns green and a synthesized trumpet fanfare sounds.
Thanks to this combination of visual staff notation, tactile keyboard input,
and immediate acoustic rewards, this website ensures rapid learning success.
It is the ideal companion for your daily musical warm-up.
